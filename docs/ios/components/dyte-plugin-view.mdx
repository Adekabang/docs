# DytePluginView
## Overview
A view which show individual (plugin/screen share) view. On top of this view we are showing instance of 
[ActiveHorizontalListView](/ios/components/dyte-active-horizontal-list-view).
In case of Mutliple(Count more than 1) plugins or Screen share, We are showing `ActiveHorizontalListView`
else it will be hidden.

Below is the image of plugin view with 3 plugins. You can select one plugin at a time.
 Now plugin name: "DocShare".

## Topics:

### Create plugin view
```swift
init(videoPeerViewModel: VideoPeerViewModel)
```
#### Parameters  
**videoPeerViewModel** Instance of type [VideoPeerViewModel](/ios/Helper-Classes/dyte-plugin-view-model)

### Adding plugins button
```swift
func setButtons(buttons: [PluginScreenShareTabButton],  selectedIndex: Int?, clickAction:@escaping(PluginScreenShareTabButton, Bool)->Void)
```
#### Parameters
**buttons:** Array of PluginScreenShareTabButton to be shown inside [ActiveHorizontalListView](/ios/components/dyte-active-horizontal-list-view)  
**selectedIndex:** Index of plugin button which needs to be selected from `buttons: [PluginScreenShareTabButton]`  
**clickAction:** Call back of type `(PluginScreenShareTabButton, Bool)->Void`.<br/>  *First parameter* is the button selected by user
.<br/>  *Second parameter* is bool that tells whether button tapped is plugin or either screen share button.
