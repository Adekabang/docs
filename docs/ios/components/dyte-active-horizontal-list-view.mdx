# ActiveHorizontalListView
## Overview
A view which show list of button of Type [PluginScreenShareTabButton](/ios/components/dyte-pluginScreenShareTab-button) shown
 horizontally


```swift
var buttons: [PluginScreenShareTabButton]
```
Array of buttons of Type [PluginScreenShareTabButton](/ios/components/dyte-pluginScreenShareTab-button) which are shown inside this list. 


```swift
func scrollToVisible(button: PluginScreenShareTabButton)
```
When there are buttons more than width of this view, Then use this method to scroll to button.
#### Parameters:
**button:** button which should must be present inside `var buttons: [PluginScreenShareTabButton]`


```swift
func setAndDisplayButtons(_ buttons: [PluginScreenShareTabButton]) 
```
This method is used to update buttons on this view. Setting buttons will reset buttons on this view.
#### Parameters:
**buttons:** Array of [PluginScreenShareTabButton](/ios/components/dyte-pluginScreenShareTab-button)
 to be shown on `ActiveHorizontalListView`
